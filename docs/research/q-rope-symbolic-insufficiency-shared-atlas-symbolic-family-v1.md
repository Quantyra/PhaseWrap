# Q-RoPE Symbolic Insufficiency Shared-Atlas Symbolic Family Memo v1

Date: 2026-03-10
Stories: S701

## Goal
Define the only remaining plausible stronger symbolic family that is still globally shared and non-lookup.

## Candidate Family
- shared-atlas symbolic regressor

## Allowed Structure
- declared coarse transition indicators
- declared first-order analog summaries
- declared pairwise cross-direction summaries
- one globally shared charting of analog space into a small fixed number of regions
- one bounded interaction layer between chart indicators and declared analog summaries

## Forbidden Structure
- latent path-state ids
- exact microstate keys
- per-chart parameters conditioned on hidden latent tuples
- unrestricted spline or kernel expansion
- arbitrary region growth chosen after looking at packet results

## Fairness Requirement
- the charting must be global and predeclared
- the same chart family must apply to every seed and every sample
- if the chart family is data-derived per latent subgroup, the review is invalid
